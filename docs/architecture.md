## Error Models

### Ionosphere (Klobuchar Model)

```
Input: Satellite azimuth, elevation, user lat/lon, GPS time
Output: Ionospheric delay (meters)

Formula:
  - Compute ionospheric pierce point (IPP)
  - Calculate slant factor F = 1 + 16(0.53 - El)Â³
  - Compute vertical delay using Î± and Î² parameters
  - Apply obliquity factor
```

### Troposphere (Saastamoinen Model)

```
Input: Satellite elevation, user height
Output: Tropospheric delay (meters)

Formula:
  - Dry component: Based on pressure and temperature at height
  - Wet component: Based on water vapor pressure
  - Apply mapping function for elevation angle
```
